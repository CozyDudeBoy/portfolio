Frontend Developer · UI / UX

사용자 경험을 설계하는 프론트엔드 개발자
정하영입니다

UI/UX를 이해하고, 단순한 화면 구현을 넘어 실제 서비스 구조까지 설계하고 구현하는 프론트엔드 개발자입니다.

React 기반 프로젝트 PING에서는 이미지 좌표를 기반으로 핀을 생성하고 질문과 답변을 연결하는 기능을 구현하며, DB 설계, API 구조, 인증 처리까지 포함한 전체 흐름을 직접 설계하고 개발했습니다. 이 과정을 통해 “기능 구현”보다 중요한 것은 데이터 구조와 서비스 흐름이라는 것을 체감했습니다.

또한 대한항공, 캐스퍼, 파스쿠찌 랜딩페이지 프로젝트를 통해 정보 구조를 재정리하고 사용자 시선 흐름을 고려한 UI 설계 경험을 쌓았습니다. 단순히 디자인을 구현하는 것이 아니라, 사용자가 정보를 더 빠르고 명확하게 이해할 수 있도록 구조를 개선하는 데 집중했습니다.

Webzen 팀 프로젝트에서는 PHP·MySQL 기반으로 게시판과 관리자 기능을 구현하며, CRUD 구조와 권한 분리, 데이터 흐름 설계를 경험했습니다.

이러한 경험을 바탕으로, 저는 “어떻게 만들 것인가”보다 “왜 이렇게 설계해야 하는가”를 고민하며 개발하는 것을 중요하게 생각합니다.

기능 구현을 넘어서 사용자 경험과 데이터 구조를 함께 설계하는 개발자로 성장하고 있습니다.
